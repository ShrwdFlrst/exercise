# Github API Search

## Requirements

- PHP 5.6+
- PHP CURL extension

## Running 

Generate a [Github token](https://github.com/settings/tokens)

    php ./php/github_search.php GITHUB_TOKEN "skin care"
    php ./php/github_search.php GITHUB_TOKEN "lipstick"
    php ./php/github_search.php GITHUB_TOKEN "mascara"
